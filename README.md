# random-user-vue

## Description

This is a simple Vue.js application that uses the Random User API to display a list of users. When a user is clicked, the right panel will appear with more information about the user.

## Technologies

- Vue.js
- Random User API
- Tailwind CSS
- Deployed on Netlify
